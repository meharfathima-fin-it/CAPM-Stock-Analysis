# CAPM Stock Analysis – Indian Equity Market

## Objective
To empirically test the Capital Asset Pricing Model (CAPM) using one year of monthly data from selected NIFTY 50 stocks.

## Methodology
- Collected 1-year monthly historical data
- Calculated monthly returns
- Computed Beta using covariance and variance
- Applied CAPM formula
- Compared expected and actual returns

## Tools Used
- Microsoft Excel
- Historical market data from Investing.com

## Key Findings
- Some high-beta stocks underperformed relative to CAPM predictions
- Significant deviation observed in certain stocks
- CAPM provides theoretical guidance but does not fully capture real-world performance
